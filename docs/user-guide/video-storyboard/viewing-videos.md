# Viewing Videos

## Overview

The Video Storyboard module provides multiple ways to view your videos, from quick thumbnail previews to full video playback with comprehensive metadata. This guide covers all viewing options and features available in the video gallery.

## Viewing Methods

### Method 1: Click to Play in Floating Player

1. **Select Video**
   - Navigate to the Video Storyboard module
   - Browse the thumbnail gallery to find your video

2. **Open Floating Player**
   - Click any video thumbnail
   - The floating video player opens automatically
   - Video starts playing automatically

3. **Use Player Controls**
   - Play/pause using the play button or spacebar
   - Seek using the progress bar or arrow keys
   - Adjust volume using mouse wheel or volume slider
   - Toggle fullscreen with F11 or fullscreen button

4. **Close Player**
   - Click the **‚úï** button or press **Esc**
   - Player closes and returns to gallery view

### Method 2: Right-Click Context Menu

1. **Right-Click Video**
   - Right-click any video thumbnail in the gallery
   - A context menu will appear

2. **Select View Option**
   - Choose **Play Video** from the context menu
   - The floating player opens with the selected video

3. **Alternative Options**
   - **View Details**: Open video details in sidebar
   - **Replace Video**: Replace with a new video file
   - **Delete Take**: Remove this video take

### Method 3: Double-Click for Details

1. **Double-Click Video Thumbnail**
   - Double-click any video thumbnail
   - Video details appear in the sidebar
   - View metadata and file information

2. **View Full Video**
   - Click **Play Full** button in the details panel
   - Opens in floating player
   - Video starts playing automatically

3. **Additional Actions**
   - **Add Take**: Add a new video take for this shot
   - **Replace**: Replace the current video
   - **Star/Unstar**: Mark as favorite or remove favorite status

## Video Metadata and Information

### Viewing Video Metadata

**Hover Preview**
- Hover over thumbnails for quick metadata
- See duration, resolution, and format
- View file size and quality indicators

**Detailed Metadata**
- Click video to view comprehensive information
- Full technical specifications
- Shot information and relationships

### Metadata Categories

**Technical Information**
- **Duration**: Video length (HH:MM:SS format)
- **Resolution**: Width x height in pixels
- **Frame Rate**: FPS (frames per second)
- **Codec**: Video compression format
- **File Size**: Storage space used
- **Aspect Ratio**: Display proportions

**Workflow Information**
- **Workflow PNG**: Status of workflow file
- **Take Type**: Final video, mask video, etc.
- **Creation Date**: When the take was added
- **File Path**: Location in project structure

**Shot Information**
- **Shot Name**: Which shot this video belongs to
- **Take Number**: Version identifier
- **Status**: Video processing and quality status
- **Relationships**: Links to related media and assets

## Troubleshooting Viewing Issues

### Common Problems

**Videos Not Loading**
- **Check File Format**: Ensure video format is supported
- **Verify File Size**: Check if files are too large
- **Check Permissions**: Ensure file access permissions
- **Refresh Gallery**: Press F5 to refresh the gallery

**Playback Issues**
- **Codec Problems**: Install appropriate codecs if needed
- **System Resources**: Free up memory and CPU
- **File Corruption**: Check for corrupted video files
- **Storage Issues**: Verify sufficient disk space

**Player Issues**
- **Graphics Drivers**: Update graphics drivers
- **Player Settings**: Reset player to default settings
- **System Resources**: Check available system memory
- **Application Updates**: Ensure AIMMS is up to date

### Error Resolution

**File Access Errors**
- **Check File Paths**: Verify video files are accessible
- **File Permissions**: Ensure read permissions are set correctly
- **File Corruption**: Check for corrupted video files
- **Storage Issues**: Verify sufficient disk space

**Playback Problems**
- **Format Compatibility**: Ensure video format is supported
- **System Requirements**: Check system meets video playback requirements
- **Memory Issues**: Free up system memory
- **Application Conflicts**: Check for conflicting applications
