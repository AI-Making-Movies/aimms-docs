# Viewing Images

## Overview

The Image Storyboard module provides multiple ways to view your images, from quick thumbnail previews to detailed full-size viewing with comprehensive metadata. This guide covers all viewing options and features available in the image gallery.

## Viewing Methods

### Method 1: Double-Click for Full View

1. **Select Image**
   - Navigate to the Image Storyboard module
   - Browse the thumbnail gallery to find your image

2. **Open Full View**
   - Double-click any image thumbnail
   - The floating image viewer will open automatically
   - Image displays at full resolution

3. **Use Viewer Controls**
   - Zoom in/out using mouse wheel or zoom buttons
   - Pan around large images by clicking and dragging
   - Fit image to window or view at actual size
   - Navigate between takes for the same shot

4. **Close Viewer**
   - Click the **‚úï** button or press **Esc**
   - Viewer closes and returns to gallery view

### Method 2: Right-Click Context Menu

1. **Right-Click Image**
   - Right-click any image thumbnail in the gallery
   - A context menu will appear

2. **Select View Option**
   - Choose **View Full Image** from the context menu
   - The floating viewer opens with the selected image

3. **Alternative Options**
   - **View Details**: Open image details in sidebar
   - **Replace Image**: Replace with a new image file
   - **Delete Take**: Remove this image take

### Method 3: Click for Details

1. **Click Image Thumbnail**
   - Single-click any image thumbnail
   - Image details appear in the sidebar
   - View metadata and file information

2. **View Full Image**
   - Click **View Full** button in the details panel
   - Opens the floating viewer with the selected image

3. **Additional Actions**
   - **Add Take**: Add a new image take for this shot
   - **Replace**: Replace the current image
   - **Star/Unstar**: Mark as favorite or remove favorite status

## Floating Image Viewer Features

### Zoom and Pan Controls

**Zoom In/Out**
- **Mouse Wheel**: Scroll up to zoom in, down to zoom out
- **Zoom Buttons**: Use + and - buttons for precise zoom control
- **Fit to Window**: Auto-scale image to viewer size
- **Actual Size**: View image at 100% pixel accuracy

**Pan Navigation**
- **Click and Drag**: Click and drag to move around large images
- **Navigation Arrows**: Use arrow buttons for precise movement
- **Overview Map**: Miniature view showing current viewport

### Navigation Features

**Previous/Next Takes**
- **Arrow Buttons**: Navigate between takes for the same shot
- **Thumbnail Strip**: See all takes for current shot at bottom
- **Quick Switch**: Jump between different versions easily

**Shot Navigation**
- **Shot List**: Navigate to different shots
- **Section Filtering**: View shots by section
- **Search Integration**: Find specific shots quickly

### Information Display

**Image Metadata**
- **Dimensions**: Width x height in pixels
- **File Size**: Storage space used by the image
- **Format**: Image type and quality indicators
- **Color Depth**: Color information and profile

**Shot Information**
- **Shot Name**: Which shot this image belongs to
- **Take Details**: Take number and creation date
- **Status Indicators**: Image status and quality markers

**Technical Details**
- **EXIF Data**: Camera and capture information (if available)
- **File Properties**: Creation date, modification date
- **Color Profile**: Color space and profile information

## Troubleshooting Viewing Issues

### Common Problems

**Images Not Loading**
- **Check File Format**: Ensure image format is supported
- **Verify File Size**: Check if files are too large
- **Check Permissions**: Ensure file access permissions
- **Refresh Gallery**: Press F5 to refresh the gallery

**Slow Loading**
- **Reduce Thumbnail Size**: Smaller thumbnails load faster
- **Clear Cache**: Clear thumbnail cache if corrupted
- **Check System Resources**: Free up memory and CPU
- **Optimize Images**: Use optimized image formats

**Viewer Issues**
- **Zoom Problems**: Check viewer settings and controls
- **Navigation Issues**: Verify image relationships
- **Metadata Missing**: Check file integrity and permissions

### Error Resolution

**File Access Errors**
- **Check File Paths**: Verify image files are accessible
- **File Permissions**: Ensure read permissions are set correctly
- **File Corruption**: Check for corrupted image files
- **Storage Issues**: Verify sufficient disk space

**Display Problems**
- **Graphics Drivers**: Update graphics drivers
- **Viewer Settings**: Reset viewer to default settings
- **System Resources**: Check available system memory
- **Application Updates**: Ensure AIMMS is up to date
